# My HTML Resume 

Hi! This is a simple, digital resume website. 

It is built using only **HTML** (the skeleton of all websites). There is no complex styling or programming logic—just simple code to display text, lists, and tables. 

It is designed to look clean and professional in any web browser.

## 🏁 How to Use This Project

You don't need any special software to run this. Just use the computer you have right now.

### Step 1: Download the Code
1.  Look for the file named 'index.html` in this repository.
2.  Download it to your computer.

### Step 2: View the Resume
1.  Find the 'index.html' file on your computer.
2.  **Double-click it.** 3.  It will open in your internet browser (like Chrome, Safari, or Edge) just like a real website.
